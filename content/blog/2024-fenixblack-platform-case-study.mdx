<Section title="The Platform" badge="Overview">

Built a complete AI marketing agency platform—**7 production applications in 4 months** (2024).

**Main Platform** (Python + NiceGUI):
- Interactive video avatars
- AI brand designer
- Research agents
- Custom React bridge for advanced UI components

**6 Satellite Micro-Apps** (Next.js on Vercel):
1. **brand.fenixblack.ai** - Brand kit generator
2. **backgrounds.fenixblack.ai** - Zoom background creator
3. **holidays.fenixblack.ai** - Holiday video generator
4. **restore.fenixblack.ai** - Photo restoration & animation
5. **canvas.fenixblack.ai** - AI hand-drawing & animation
6. **growth.fenixblack.ai** - Credit management

**Live:** [fenixblack.ai](https://www.fenixblack.ai)

</Section>

<Section title="Technical Innovation" badge="Architecture">

**Main Platform:**
- Built with **Python + NiceGUI** for rapid AI workflow prototyping
- Created **custom React bridge** to use React components within Python
- Integrated multiple AI models for different creative tasks
- Video avatar system for interactive presentations

**Micro-Apps:**
- Each specialized app deployed independently on Vercel
- Next.js for fast, responsive user experiences
- AI model integration: OpenAI, Stable Diffusion, others
- Credit-based usage system
