Built a production-ready enterprise AI chat widget from scratch in **3 weeks** (October 2024).

**Core Features:**
- RAG pipeline for brand-specific responses
- Voice mode powered by Gemini Live
- Video calls with Daily.co integration
- Meeting transcription via AssemblyAI
- Sub-200ms response times from brand data
- Website generator
- Automated content scraping

**Live:** [okidoki.chat](https://okidoki.chat)

</Section>

<Section title="Technical Architecture" badge="How It Works">

**Stack:**
- **Frontend:** Next.js 14 + TypeScript
- **AI:** Gemini Live (voice), Groq (fast inference)
- **RAG:** Custom pipeline with vector storage
- **Video:** Daily.co WebRTC
- **Transcription:** AssemblyAI
- **Infrastructure:** Vercel Edge Functions + Redis
- **Caching:** Redis for sub-200ms responses

**Key Innovation:** Pre-processing RAG content and intelligent caching moved heavy computation to build time, enabling real-time response speeds.
